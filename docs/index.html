    About The Project

    NON-GON is a curated library of analytical functions for computing shortest distances and proximity queries between smooth convex shapes in real time. Designed for applications in game engines, physics-based simulations, and interactive systems, NON-GON focuses on non-polygonal geometriesâ€”offering a high-precision, memory-efficient alternative to mesh-based approaches.
